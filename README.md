# @dateforge/react-popover

Composable popover primitives for React.

`@dateforge/react-popover` is a small overlay primitive built for custom pickers, dropdowns, floating panels, and Dateforge components.

It follows the Dateforge composition model: a lightweight root wrapper plus separate modules that can be imported from the root or through subpath imports.

## Status

Experimental. API may change before the first stable release.

## Install

```bash
npm install @dateforge/react-popover
```
